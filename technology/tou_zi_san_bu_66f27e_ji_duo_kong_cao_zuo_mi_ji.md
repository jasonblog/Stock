# 投資三部曲~及多空操作秘笈


<div id="content_all" style="padding : 1px 0 15px; *height : 1%"><p>投資三部曲~及多空操作秘笈 <br>． 投資紀律篇 </p><p>． 投資趨勢篇 </p><p>． 投資策略篇 </p><p><br>&nbsp; <br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; 投資紀律篇 <br>&nbsp; 淺談操作金融商品的? 專精、 紀律 、停損 的觀念 </p><p><br>&nbsp; 一. <br>『專業』 <br>是指專一的工作，但並非完全是學有專精的一種工作；操作金融商品不能只是專一的工作，而是需要學有專精的一種工作，您若想有學有專精的水準，您勢必需要把它當成專一的工作；贏家一定是專家，專家不一定是贏家，大多數的技能，別人都可能教你，唯獨成為贏家的技能，別人不可能教你，因為他把諸位都教成贏家，那麼他怎麼能贏到你們的錢呢 ? 請各位看倌大人評斷一下，電視上的名嘴或報章雜誌上的名筆，他們是贏家或是專家 ? </p><p>在這個市場裡，有個永不變的法則，穩賺不賠是開賭場的政府，保證賺錢的是印股票的公司老闆及證券公司，散戶在股市中有幾個能真正賺到錢 ? 無法擁有分析判斷的能力或沒有準備好 ，就貿然進場操作，猶如盲人騎馬，危矣 ! 殆矣 ! </p><p>以金融商品操作當職業不是一件簡單的事，除非您能掌握住行情之趨勢，以及具備該商品之專業知識，並要建立一套完全屬於自己的操作模式，做一個完整的規劃，而不是隨意投入市場就可以隨意賺錢 ! 只憑聽聽電視股市分析或閱覽一下報紙報導 ，就可賺到錢，那麼還有誰願意每天辛苦地上班八小時，甚至去做粗工呢 ? 講白一點，盤中看盤就能讓你買到低點賣到高點，天底下哪有這樣輕鬆的事 ? 如果有，那些每天到號子看盤的菜籃族早就賺爆 ? <br>&nbsp; </p><p><br>&nbsp; 二. <br>『遵守紀律依法操作』 <br>任何一個時間點，一定有很多人看空，也一定有很多人看多；重要的是，當方向錯時，要懂得修正看法 ( 停損出場 )，看錯時加碼只會讓自己愈陷愈深，當趨勢對時，要借力使力 ( 順勢加碼 ) 以求最大的獲利 ! </p><p>行情盤勢上下區間震盪，在操作上存在著不可避免的盲點，隨時有追高殺低的可能，此時只能期待等方向趨勢出現，再圖追求更大的獲利契機，我們在此市場裡猶如『又聾又啞又文盲的人』，我們要如何搭上將開動且方向又正確的火車呢 ? 火車未動之前我絕不上車，火車開動了，我馬上跳上車，但方向不對要趕快下車，否則您想到台北，它卻把載到高雄 ! </p><p>波段的低 ( 谷 ) 點通常出現在市場最悲觀的時候，也就是作多方者已經虧損到無法忍受時，波段的高 ( 峰 ) 點通常出現在市場最樂觀的時候，也就是作空方者已經虧損到無法忍受時，因此這兩個點﹝低 ( 谷 ) 、高 ( 峰 )﹞就是絕佳的買進點或賣出點。規劃是降低忙亂進出的原則，除此之外就是等待方向的趨勢的形成 ! </p><p>例如您有 100萬資金，您該怎樣去規劃操作呢 ? </p><p>假設您設定 1年想要賺 20萬，每筆交易只能賺 5000，只能賠 3000，成功的比例是60%，失敗的比例是 40%，假設一年總交易筆數為 K，則得到以下方程式 </p><p>( 5000 × 60% × K ) – ( 3000 × 40% × K ) = 200000，k = 111 </p><p>代表您整年的交易筆數不得超過 111筆，也就是說每月交易不得超過 111 / 12 = 10筆，這10筆中，您必須成功 6筆，失敗 4筆，丟一次銅板猜出正反面的機率是 1 / 2，但連續丟 5次都猜對的機率卻只有 1 / 32 而已，股票交易過程中，不論買進或賣出，都得支付 0.1425%的手續費，而賣出後，還要繳交 0.3%的證交稅，算一算每進出一次，成本約為 0.6%。用絕對金額來表達，如果一次買賣股票的數額為 10萬元，那麼成本就是 600元。 </p><p>讓我們來計算一下，每週進出股市一次侵蝕獲利的程度。以一年 52週計算，交易成本約 30%，It means ( 意思是指 )，10萬元每週在市場轉一次，一年就得平白繳交給券商及政府 3萬元，相對的，要 cover 這 3萬元的成本，你必須有能力每年賺進 30%的獲利。現在靜下來想想，每年賺 30%，你有這樣的本事嗎？若沒有，那麼您就別整天殺進殺出吧！若沒經過規劃判斷地進出頻繁，您就是在當券商及政府的義工罷了 ! <br>&nbsp; </p><p><br>&nbsp; 三. <br>『停損』 <br>是避免受傷過大的法寶，它絕對需要在學習技術分析之前，就先建立的觀念，要先會去執行的一種保命動作，並且一定要嚴格遵行，除非您是已經知道波段高、低點在哪裡的人 ! 若是您還沒辦法達到『賣在最高，買在最低』這樣境界的人，不會停損或不想停損，離開市場是早晚的事。 </p><p>當你買進一種金融商品時，應該事先就已做了一些功課和預期，包含價位 、進場時機，甚至出場部位等，當事實與預測的相反時，豪不猶豫的停損，代表之前的推測依據是錯的，這時已犯了第一次錯誤，如果到了可忍受的風險臨界點還不停損，這時已犯了第二次錯，不停損，會讓錯誤越來越大，最後危害到自己的資本，能忍受犯幾次錯呢..... ? </p><p>『停損』是在看錯行情時，面對錯誤的表現，若判斷錯誤又不肯認錯，存在著『停損，是認輸，就是輸錢』的觀念 ! 期待著股價、指數遲早會回來，殊不知若不停損會輸更多，大多數人寧可套牢，就是不願意停損，一直認為因停損而輸錢 ，是一件不能接受的事，而把錢輸在市場，是可以接受的 ! </p><p>『停損』是技術分析學習者的根基，是護身盔甲和盾牌，市場的新兵沒有準備這些裝備，一定畢業得很快，市場的老手自仗武藝高強，脫盔卸甲、放下盾牌，遲早中箭落馬，『停損』是在學習到神算之前的保命符。 </p><p>有些人說經常『停損』，錢也會輸完，若會慘到停損把錢輸完，是停損的問題嗎 ? 該是操作技術有問題吧 ! 該是您沒辦法抓到正確的買賣點吧 ! 先學會停損，再談學習技術及專業分析，最後才是建立一套屬於自己的東西，這是贏家操作的三步曲。 <br>&nbsp; </p><p><br>&nbsp; <br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; <br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; <br>&nbsp; 贏家的十個觀念 </p><p><br>&nbsp; 一. <br>時機： </p><p>開盤或交易進場前務必注意『時機』，有進場 ( 買進或放空 ) 的行為 ，一定有出場 ( 賣出或回補 ) 的行為，適當的進場點，那麼出場就是賺多或賺少的情況，不適當的進場點，那麼出場就是賠多或賠少的情況，『寧可不賺，千萬不可貿然進場』，『寧可錯失機會，千萬不可失去鈔票』，並非隨時都具備符合【多】或【空】的條件，因此等待是必須的。 </p><p><br>&nbsp; </p><p><br>&nbsp; 二. <br>漲跌： </p><p>【漲】是為了之後的回跌，【跌】是為了之後的回升，波段漲勢已結束，但是大多數的人，會不自覺地以為漲勢還會再繼續，波段跌勢已結束，但是大多數的人 ，會不自覺地以為跌勢還會再繼續，因此每一次都要以「準備」的心情，在『轉折點』做【買】或【賣】的動作，所謂『轉折點』即為峰或谷的循環已完成，再加上平均線或 KD線的交叉確認訊號的出現，即【買】要買在跌不下去的『轉折點』，【賣】要賣在漲不上去的『轉折點』。 </p><p><br>注意： <br>一、多頭排列時：價位由上往下時觀察『支撐』。 <br>二、空頭排列時：價位由下往上時觀察 『壓力』。 </p><p><br>&nbsp; </p><p><br>&nbsp; 三. <br>兩極： </p><p>【漲勢將盛極而衰】，【跌勢將否極泰來】，有贏家一定有輸家，因此贏家與輸家是互相對立的，在一個波段的峰極，一定是利用誘多、養多再殺多，在一個波段的谷極，一定是利用誘空、養空再軋空，如果您想成為市場中的贏家，一定要設法知道目前盤勢的位置是在哪裡 ? 谷底、半山腰或山頂在谷底摔下來 ，僅是滿身酸痛，在半山腰摔下來，則是骨頭斷裂，在山頂摔下來，必定是粉身碎骨矣。 </p><p>&nbsp; </p><p><br>&nbsp; 四. <br>口數： </p><p>符合進場條件時，先以【一單位】進場試單，當試單成功時 ，再以【二單位】順向加碼，當趨勢確認時，仍可以「菱形」 的方式，再以【一單位】順向加碼，注意：所謂「順向」是指在倉部位皆在獲利之中，否則即為「逆向」，「逆向」絕不可做【加碼】，方向已知不對，務必要做【停損】的動作。 </p><p>&nbsp; </p><p><br>&nbsp; 五. <br>進出： </p><p>進場後尋找出場的時機，出場後尋找進場的時機，在盤中或進場後，務必在意【進出】要領，贏家的特質就是在【進出】的動作上，有果斷的能耐，該斷就斷，該加碼就加碼 ( 但不可以在不對的情況加碼攤平 )，例如：做多時，已形成空頭排列決不可加碼攤平；做空時，已形成多頭排列決不可加碼攤平，進場或出場一定要『有根有據』，憑感覺或碰運氣 ，都不是正確的操作行為，終究將被市場所淘汰的。 </p><p>&nbsp; </p><p><br>&nbsp; 六. <br>賺賠： </p><p>先計算將賠多少錢，再計算將賺多少錢，操作任何金融商品，一定要先評估您可以賠得起多少錢 ? 操作的正確觀念是【風險】重於【利潤】，如果能夠把【風險】擺在第一優先的考量，那麼操作的【風險】就可以控制在最小的範圍內，贏家和輸家的最大的差異就是「觀念」的不同，簡言之，輸家的「觀念」是先計算將賺多少錢，再計算將賠多少錢。 </p><p>&nbsp; </p><p><br>&nbsp; 七. <br>避險： </p><p>要先設法避開風險，才有機會賺取利潤，吾人在室外或陽台曬衣服，一旦陽光消失了，接著出現烏雲密佈，不管衣服是否已曬乾，一定需要先把衣服收起來，因此無論【作多】或【作空】，一旦順向的條件消失了，接著出現確認訊號，不管有無利潤，一定需要先做應變，因為此時的風險已逐漸昇高矣 ! </p><p>&nbsp; </p><p><br>&nbsp; 八. <br>停損： </p><p>順向的條件無法由盤勢做確認時，就要執行【停損】，要做到【停損】，一定要先去除【成本觀念】的控制，假如你的觀念被【成本】綁住，欲執行【停損】的動作談何容易，該認賠時就要迅速認賠，記住：最初的損失就是最小的損失，至於虧損多少並不是重要的，若無法嚴守操作紀律，將註定是輸家。 </p><p>&nbsp; </p><p><br>&nbsp; 九. <br>靈活： </p><p>多空走勢要明確，進場出場要主動，多空的節奏就是一種慣性，低點不再，屢創新高為【漲勢】的軌道，高點不再，屢創新低為【跌勢】的軌道，只是勤做功課，並且用心地觀察及體會，必然有多空的節奏，您目前所持有的部位，該留多單或留空單或空手，是由您主動地去決定，而不是被盤勢所逼再做決定。 </p><p>&nbsp; </p><p><br>&nbsp; 十. <br>對錯： </p><p>賺錢時對在哪裡 ? 賠錢時錯在哪裡 ?【計劃】、【執行】、【檢討】是贏家的必要手段，金額操作一定要知其所以然，【檢討】是每天必做的功課，務必要養成【檢討】的習慣，任何一筆交易的結果，無論是對或是錯，都要進行【檢討】的動作，要牢記每次做對而賺錢的原因，來做為重複獲利的模式，更要牢記每次做錯而賠錢的原因，警惕自己日後不再犯同樣的錯誤。 </p><p>&nbsp;</p><p>投資趨勢篇 </p><p><br>&nbsp; <br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; <br>&nbsp; 傳統趨勢分析 <br>道氏理論 </p><p><br>&nbsp; 它是一種最古老、最著名的技術分析方法，由美國《道 Don》先生首創。趨勢理論所研究的是整個市場的走向趨勢，許多的指標被創造出來，都是用來對趨勢發生之後『 加以確認 』的工具，以及觀察支撐與壓力的效用，來判斷趨勢的方向，所以趨勢分析是技術分析中最重要的方法，也是其他分析方法的基石。 <br>道氏趨勢理論的假設： <br>1.&nbsp;&nbsp; 走勢存在著某種趨勢進行，它是一種慣性，在支撐區和壓力區有可能使這種慣性改變或終止，在沒有事實來確認趨勢改變之前，應假定趨勢仍未改變。 <br>2.&nbsp;&nbsp; 趨勢終將週而復始，重複發生。因此觀察過去的趨勢，可發現走勢確實存在著某種重複發生的情況，主要是因為人類行為本身便具『重複性』所致。 <br>『道氏理論』認為趨勢的走勢三種運動﹕ <br>就整個趨勢來分析，觀察過去的歷史資料，可將整個趨勢的循環分為三種走勢，分別為主要趨勢、次級趨勢及小型趨勢。這三個趨勢分別包含了三種不同的架構，並且一直重複循環不已。 <br>(1)主要趨勢：&nbsp;&nbsp; 巿場出現中期看漲趨勢或中期看跌趨勢。 <br>( 於峰谷循環裡，所形成的 1 ~ 3 波段的峰循環或谷循環 ) 主要趨勢是反應上升或下降的走勢，當每一波漲勢的高點突破前波高點，且中間夾著的回檔也愈來愈高時，則將形成多頭 ( 上升 ) 的趨勢。而當跌勢的低點跌破前波低點，且中間夾著的反彈也愈來愈低時，則將形成空頭 ( 下降 ) 的趨勢。 </p><p>(2) 修正趨勢或折返趨勢： <br>&nbsp;&nbsp;&nbsp; 它是在股價上漲的趨勢中出現一小波段下跌的運動。 <br>或是在股價下跌的趨勢中出現一小波段上升的運動。 <br>當主要趨勢發生過度延伸的情況時，股價將產生與主要趨勢反向變動的暫時性折返現象，這便是修正趨勢或折返趨勢。發生次級折返趨勢的主要原因為獲利回吐賣壓，即投資人對第一波中期漲勢的過度樂觀等因素造成的。次級折返趨勢都是較難辨識的，有時折返幅度過大時，容易誤以為主要趨勢已發生改變。 <br>掌握次級折返趨勢的特徵，才不至於迷思於折返趨勢中。次級折返趨勢的低點應該要高於前一個折返的低點，以保持波谷間的連線為上升趨勢線。折返的幅度與主要中期走勢的幅度有關，主要中期走勢較小時，則表示市場信心不夠強，折返走勢的幅度將較大，反之亦然。需注意一點，即次級折返走勢不一定是以股價修正的型態出現，它可能會以橫盤整理來取代拉回。 </p><p>(3)小趨勢：&nbsp;&nbsp; 就是走勢的每天變動，受到每日的小波動，構成短線上的小型趨勢。 <br>這種小型趨勢屬於短暫性的價格現象，它的主要的成因是由於交易行為所造成的波動。每段主要、次要趨勢，均存在著微小波動，即小趨勢。一般在作投資決策時，小型趨勢是一個可以忽略的走勢。 </p><p>&nbsp; </p><p><br>一個主要循環 </p><p><br>&nbsp; 1.&nbsp;&nbsp; 一個主要循環包含一個主要《上升》趨勢 和 主要《下降》趨勢。 <br>2.&nbsp;&nbsp; 一個主要《上升》或《下降》趨勢約含 1 ~ 3 個 ( 即約含 2.5個 ) 中期走勢。 <br>3.&nbsp;&nbsp; 每個中期走勢包含一個主要中期走勢 和 一個次級折返趨勢。 <br>4.&nbsp;&nbsp; 每段主要、次要趨勢，均存在著微小波動，即小趨勢。 </p><p><br>&nbsp; </p><p><br>壓力與支撐 </p><p>&nbsp; 支撐的意義即為需求集中的區域，亦即潛在的買進力道的聚集區；而壓力則為供給集中的區域，價格達到該區域時，將引發賣方 ( 空方 ) 力量出現。 <br>趨勢線是未來價格的可能走向，因此在上升趨勢線的下緣，存在著潛在不小的買進力量 ( 支撐 )，使得價格持續往趨勢線方向走。在下降趨勢線的上緣，存在著為數不小的賣出力量 ( 壓力 )，使得價格有沿著下降趨勢線進行的傾向。 <br>支撐或壓力常出現的位置： <br>a.&nbsp;&nbsp; 過去發生的波峰或波谷，將是壓力或支撐的區域。 <br>b.&nbsp;&nbsp; 平均線及趨勢線皆為支撐及壓力的所在處。 <br>c.&nbsp;&nbsp; 短線過熱將形成心理關卡，為支撐或壓力的所在處。 <br>d.&nbsp;&nbsp; 整數價位亦為支撐或壓力之所在處。 <br>壓力就是【多數人套牢的價位或位置】， <br>支撐就是【多數人買進的價位或位置】。 <br>走勢遇到壓力點或區域，對多方而言，將會有較大的抵抗，但不一定就不會被突破。走勢遇到支撐點或區域，對空方而言，將會有較大的抵抗，但不一定就不會被跌破。 <br>金融商品的買賣操作，絕不可用價位比較法或價位參考點法，這是不適當的方法，失誤的機率很大，正確的方法是在波段循環完成之後，在其壓力或支撐的用位置使用觀察法，因為您預期的價位不一定會到達，而位置一定會到達，即使您預期的價位到達了，該價位並不一定發生壓力或支撐的效用，因此吾人應觀察該壓力或支撐有無被突破或跌破，做為進場或出場之依據。 <br>心理面必須搭配技術面，如此才能正確加解讀市場目前所在位置，是在走多頭還是在走空頭，目前所在位置是在何處 ? 要判斷市場循環，需要以足夠的技術功力，搭配目前市場的心理面，如此才能精準研判出現在市場到底在哪一階段 ! <br>&nbsp; </p><p>&nbsp;</p><p>投資策略篇 </p><p><br>&nbsp; <br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; <br>&nbsp; 傳統式技術指標分析 <br>移動平均線 </p><p><br>&nbsp; 移動平均線 ( Moving Average Line ) 之定義及範圍 <br>1. 移動平均線 <br>( MA線 )： 移動平均線是用統計的方式，把欲平均的天數，每天收盤價以累積加總後，除以其平均天數，得出一平均值，然後把它連接成一條線，用來觀察價格趨勢。可根據價格與平均值的排列順序，來研判目前位置是買進點或賣出點。移動平均線的意義，代表某期間內投資人的平均成本，有判斷價格的趨勢、買賣點及設定停損 ( 利 ) 的功用，所以從移動平均線和價格之間的關係，可以看出價格即將產生變動的方向。 </p><p>2. 多頭與空頭的排列 <br>&nbsp; A.&nbsp;&nbsp; 多頭排列：短期平均線 &gt; 中期平均線 &gt; 長期平均線。 <br>即由上而下依序為：價格、短期平均線、中期平均線、長期平均線。 <br>移動平均線 ( MA線 ) 呈多頭排列時，表示回升行情開始，注意：多頭排列的上升趨勢路徑途中，別被暫時的修正或回檔的枯景所蒙蔽，這些暫時的景象，對空方而言，它是包著毒藥的糖衣，這是【誘空】的陷阱。 <br>黃金交叉 ( 多頭走勢 ) <br>1.&nbsp;&nbsp; 短期平均線 ( 例：5日MA線 ) 向上突破中期平均線 ( 例：10日MA線 )，形成了【黃金交叉】，這是做多者的『第 2個』買進時機。 <br>2.&nbsp;&nbsp; 短期平均線 ( 例：5日MA線 ) 向上突破長期平均線 ( 例：20日MA線 )，形成了【黃金交叉】，這是做多者的『第 3個』買進時機。 <br>3.&nbsp;&nbsp; 中期平均線 ( 例：10日MA線 ) 向上突破長期平均線 ( 例：20日MA線 )，形成了【黃金交叉】，這是做多者的『第 4個』買進時機。 </p><p><br>&nbsp; B.&nbsp;&nbsp; 空頭排列：短期平均線 &lt; 中期平均線 &lt; 長期平均線。 <br>即由上而下依序為：長期平均線、中期平均線、短期平均線、價格。 <br>移動平均線(MA)呈空頭排列時，表示下跌行情開始，注意：空頭排列的下降趨勢路徑途中，別被暫時的修正或反彈的榮景所蒙蔽，這些暫時的景象，對多方而言，它是包著毒藥的糖衣，這是【誘多】的陷阱。 <br>死亡交叉 ( 空頭走勢 )： <br>1.&nbsp;&nbsp; 短期平均線 ( 例：5日MA線 ) 向下跌破中期平均線 ( 例：10日MA線 )，形成了【 死亡交叉 】，這是做空者的『第 2個』賣出時機。 <br>2.&nbsp;&nbsp; 短期平均線 ( 例：5日MA線 ) 向下跌破長期平均線 ( 例：20日MA線 )，形成了【 死亡交叉 】，這是做空者的『第 3個』賣出時機。 <br>3.&nbsp;&nbsp; 中期平均線 ( 例：10日MA線 ) 向下跌破長期平均線 ( 例：20日MA線 )，形成了【 死亡交叉 】，這是做空者的『第 4個』賣出時機。 </p><p><br>移動平均線 ( MA線 ) 的特性及形態： <br>移動平均線 ( MA線 ) 有四種特性： <br>1.&nbsp;&nbsp; 趨勢： 因移動平均線是由某期間內的價格漲跌平均計算，當走勢上漲時，此線之扣抵值將加上更高值，使得此線形成上升的趨勢，當走勢下跌時，此線之扣抵值 ，將加上更低值，使得此線形成下降的趨勢，因此計算扣抵值，可預估移動平均線未來是繼續上升、走平或向下彎頭。 <br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; <br>2.&nbsp;&nbsp; 安定： 移動平均是由移動中連續數日之收盤價平均計算而成，如以 5日移動平均線為例，如果扣抵後上漲 50點，平均線只上揚 10點，因此平均線比價格線較平穩，並且平均線的基期設定越長期，則該平均線的曲線越平穩。 <br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; <br>3.&nbsp;&nbsp; 助漲： 價格從平均線下方向上突破，使得平均線也開始上漲，代表買進平均成本的提高，而成本的上移代表多頭支撐線向上移，當價格回跌至平均線附近，將會產生支撐的功用，此位置為買進時機，當價格於一個區間整理時，平均線會由上升轉變為走平，當扣抵值為負時，則移動平均線將會向下彎頭。 <br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; <br>4.&nbsp;&nbsp; 助跌： 價格價從平均線上方向下跌破，平均線也開始由走平轉變為向下降，平均線開始成為多頭的上檔壓力線，股價反彈到平均線附近，自然產生壓力，當價格跌落於平均線之下，代表是處於虧損狀況，再度回成本價時，便成為「先求保本」的賣出時機。 <br>移動平均線之『 減加值 』的關係： <br>今 日 值 減 N 日 前 值 移 動 平 均 線 <br>大 正 值 上 升 <br>小 正 值 微 升 <br>平 值 走 平 <br>小 負 值 微 降 <br>大 負 值 下 降 <br>移動平均線 ( MA線 ) 之類型 <br>在短期日線中較常用的是：5日、10日、20日 MA 線之組合。 <br>在中期週線中較常用的是：5週、10週、20週 MA 線之組合。 <br>在長期月線中較常用的是：5月、10月、20月 MA 線之組合。 <br>移動平均線之優缺點 <br>優點： 1.&nbsp;&nbsp; 可表現出價格的趨勢方向，亦可以比較短、中、長期的平均成本。 <br>2.&nbsp;&nbsp; 平均線的分析方式比較簡單，使投資者能清楚地瞭解目前價格的動向。 </p><p>缺點： 1.&nbsp;&nbsp; 短期日線極為敏感，並且可靠性不夠，所以常常會出現許多錯誤判斷點。 <br>2.&nbsp;&nbsp; 長期日線則較具有可靠性，但相對其敏感度卻不夠，所以當買賣點訊息發出時，價格早已反應了一段時間，所以獲利將會受限。 <br>3.&nbsp;&nbsp; 移動平均線變動緩慢，不容易把握股價趨勢的高峰與低谷。 <br>4.&nbsp;&nbsp; 在價格波幅不大的盤整期間，出現上下交錯的買進或賣出訊號，除非使用 3日移動平均線 ( 3MA ) 對應 6日移動平均線 ( 6MA ) 的『乖離』，否則將讓使用者無法決定此位置「該買或該賣」。 </p><p>K-D指標 ( 又稱隨機指標 ) <br>1.&nbsp;&nbsp; KD指標通常在 80以上為超買區，在 20以下為超賣區，且 KD指標訊號明顯，適合短線操作，是短期分析指標，常被用於日線圖。 <br>2.&nbsp;&nbsp; 由於 D值較 K值平緩，因此當 K值在超賣區向上突破 D值時，表示趨勢發生改變，( 即 KD指標低檔區形成黃金交叉向上時 )，為買進訊號；而當 K值在超買區向下跌破 D值時 ( 即 KD指標高檔區形成死亡交叉向下時 )，則為賣出訊號。 <br>3.&nbsp;&nbsp; 背離形成為趨勢轉變訊號： <br>注意：正常狀況下，走勢突破前波高點，KD指標理應突破前波高點； <br>走勢跌破前波低點，KD指標理應跌破前波低點。 <br>A.&nbsp;&nbsp; 當走勢沒有突破前波高點時，但 KD指標已經突破前波高點，這樣的背離現象稱為『正背離』，當正背離發生時為技術面【轉強】的徵兆。 <br>當走勢突破前波高點，但 KD指標並沒有突破前波高點時，這樣的背離現象稱為『負背離』，當負背離發生時為技術面【轉弱】的徵兆。 <br>B.&nbsp;&nbsp; 當走勢跌破前波低點 ，但 KD指標並沒有跌破前波低點時，這樣的背離現象稱為『正背離』，當負正離發生時為技術面【轉強】的徵兆。 <br>當走勢卻沒有跌破前波低點時，但 KD指標已經跌破前波低點，這樣的背離現象稱為『負背離』，當負背離發生時為技術面【轉弱】的徵兆。 <br>優點： 1.&nbsp;&nbsp; KD指標發生黃金交叉或死亡交叉時，買進或賣出訊號很明確。 <br>2.&nbsp;&nbsp; KD指標與價格在超買區或超賣區形成背離，可判定原走勢趨勢轉弱或轉強。 <br>3.&nbsp;&nbsp; KD指標將改善移動平均線反應遲鈍的缺點。 </p><p>缺點： 1.&nbsp;&nbsp; 在長期上漲的趨勢時，KD指標經常會在超買區高檔徘徊，亦即當價格到達超買區時，可能會向上再延伸一段行情，因此在大多頭行情，容易錯失所延伸的那段行情。 <br>2.&nbsp;&nbsp; KD指標較敏感，買賣訊號出現較頻繁，尤其在盤整期間或箱型區時。 <br>3.&nbsp;&nbsp; 可能出現多次交叉騙線，使投資人錯誤地買進或賣出，因而增加交易成本。 </p><p><br>&nbsp; </p><p><br>亞當理論 (Adam Theory) </p><p><br>&nbsp; 1.&nbsp;&nbsp; 賠錢的部位絕不要”加碼攤平”。 <br>如果你目前操作的是賺錢的部位，你預期的趨勢方向是對的。如果你操作的是賠錢的部位，那麼你預期的趨勢方向是錯的。如果你已經錯了，又增加錯誤的部位，這樣的話，將使你錯得比目前更離譜。 </p><p>2.&nbsp;&nbsp; 在開始操作或加碼時，務必要同時設停損，以便在您萬一差錯時，你能出場。 <br>在您開始進場操作之前 ( 無論是作多或作空 )，必須先決定這筆操作您【願意錯多久】及【願意賠多少錢】? 因為只有在進場之前，才能作出客觀的決定。一旦您處在市場之中，你的決定就不再客觀了。 </p><p>3.&nbsp;&nbsp; 除非是朝操作所要的方向，否則絕不取消或移動停損。 <br>你會想朝操作反方向移動停損的唯一時刻，是操作部位發生虧損，而且市場對你不利時移動停損位置，導致操作賠更多錢。如果你移動停損，那麼你的期待將完全覆蓋了你的理性和技能。 </p><p>4.&nbsp;&nbsp; 絕不讓合理的小損失演變成一發不可收拾的大損失。 <br>方向不對，立即停損，大損失將會在幾天內賠掉您一個月所賺的。為什麼不接受小損失，而寧願任憑它演變成大損失，若一筆大損失約等於十筆小損失，您不可能在適當的買賣區域裡，做十次的『試單』動作，沒有一次成功的吧 ? 記住：被針刺幾次是不太可能致命的；深長的傷口才會要人命。 </p><p>5.&nbsp;&nbsp; 不要同時擁有太多不同的部位。 <br>由於部位很多，即使停損點很接近，許多或所有部位對你不利，仍可能造成重大虧損的。由於部位太多，可能發生難以照顧之情況，由於擁有的部位過於分散，在適當時機無法做分批加碼。任何時刻都不要冒虧大錢的險，只要趨勢、策略和紀律都能正確，戶頭內資金動用到一半，已經能達成您預期的操作利潤了，所餘之一半資金應該當成承擔風險之用。 </p><p>6.&nbsp;&nbsp; 別去抓頭部和底部，讓市場把它們抓出來。 <br>市場偶爾會讓你抓準一兩次頭部和底部，為什麼不只等反轉確立呢 ? 為什麼要不願意丟掉確立之前的所有利潤呢 ? 這全是貪婪在作祟，你有多少次因為不肯等候而賠了錢 ? </p><p>7.&nbsp;&nbsp; 別擋在列車前面。 <br>如果市場往某個方向爆炸性發展，千萬別逆勢操作，除非有強烈的證據，顯示反轉也已發生。超買的市場絕沒有不能再漲的理由。賣超的市場絕沒有不能再向下的理由。這是敵人喜歡佈置的陷阱。擋在列車前面，您將承受極大的風險。 </p><p>8.&nbsp;&nbsp; 保持彈性。 <br>亞當理論可能會錯，任何事情可能偶爾出差錯。記住亞當理論所說的是機率很高的事，而不是絕對肯定的事。您所期望的是或然率，而不是絕對值。 </p><p>9.&nbsp;&nbsp; 操作不順時，不妨縮手休息。 <br>如果你一再發生虧損，請退場讓你的情緒冷靜下來，等頭腦變得清醒再說。多數操作者之所以會賠錢，其中一個理由是，不受約束的途徑走起來最輕鬆。當你的資金暴跌時，要放手一段時間是件相當難的事。你不願承認自己所做的每件事都錯了。你只會告訴你自己，操作還沒結束，只是一時的挫敗。以前你也曾經迅速扭轉乾坤，所以可以再來一次。現在就不承認失敗，而且從頭再來時，本錢會少很多，操作者處於劣勢。由於操作者受到很大的壓力，必須迅速反敗為勝，所以他比較難以保持客觀的態度。他會冒平常時候自己不肯冒的險。 </p><p>10.&nbsp;&nbsp; 問問你自己，若您想從市場中賺一筆錢，你遵循依據的方法是什麼 ? <br>如果你在市場上操作的真正的理由是想賺錢，那麼遲早你會知道，一個人能不能從市場上賺錢，取決於他有沒有遵守這十大守則。至於他賺多少錢，則取決於他進場和退場的方法。你將瞭解不會只因為使用亞當理論或其他任何方法，就可以成為贏家。亞當理論只是給你一個進場的理由。這個理由是：市場有很高的機率，往某個特定方向移動一段時間。 </p><p>策略評論 ? <br>傳統式指標只是統計學的概念，一般來說都是落後指標，它們都是價格之走勢上升了或下降了之後，指標才會反應，所以，指標是告訴我們這段時間走勢是強勢還是弱勢，而不是告訴我們未來會如何走，或目前走勢所處的位置在哪裡 ? 報章雜誌或電視第四台的分析師，賣膏藥的居多，尤其自誇倍數獲利者，一定是假貨，真能如此，一個月能賺本錢的一倍，那麼他們幹啥跑到螢光幕前聲嘶力竭，鬼吼鬼叫 ? <br>加碼攤平 ? <br>1.&nbsp;&nbsp; 在《非適當》的位置買進作多或賣出作空時，虧損的部位決不可加碼攤『平』，你很容易愈攤愈『貧』! 因為您無法知道此波段的盡頭在那裡？ 再者，要跌百分之多少才加碼攤平 ? 並沒有一定的標準 ? <br>2.&nbsp;&nbsp; 在《適當》的位置 ( 即峰谷循環完成 ) 買進作多或賣出作空時，並經確認指標確認後，可以做加碼攤『成本』。 <br>3.&nbsp;&nbsp; 當您發現走勢方向與您的預期相反 ( 逆向 ) 時，您若仍然繼續往錯誤的方向走，那麼您只會往相反方向越走越遠，或許您將耗損您的時間和體力 ( 資金成本 )。 <br>追高殺低 ? <br>1.&nbsp;&nbsp; 在谷循環完成後的《 谷區 》，該追高時就該『 追高 』，因為此位置的追高，很可能是短期內的『 低進 』。 <br>2.&nbsp;&nbsp; 在峰循環完成後的《 峰區 》，該殺低時就該『 殺低 』，因為此位置的殺低，很可能是短期內的『 高出 』。 <br>逢高賣出 ? 高出低進 ? <br>1.&nbsp;&nbsp; 勿在多頭排列的上升趨勢途中『高出或逢高賣出』，走勢為上升趨勢時，您先行『高出或逢高賣出』的動作，那麼短期內 ，您可能沒有低價可買進補回。 <br>2.&nbsp;&nbsp; 應在空頭排列的下降趨勢途中『高出或逢高賣出』，若在此位置『逢高先行賣出』之後，那麼短期內 ，您才可能有低價可買進補回。 <br>3.&nbsp;&nbsp; 多頭排列的上升趨勢中的某日大跌，分析師們都會強烈建議次日『 逢高賣出 』，但是，假如某日的大跌僅是【修正】，次日的逢高賣出不就是賣出太早了嗎 ? 假如某日的大跌是【回檔】或【回跌】，次日可能沒有『逢高賣出』的機會了 ! <br>逢低買進 ? 低進高出 ? <br>1.&nbsp;&nbsp; 勿在空頭排列的下降趨勢途中『低進或逢低買進』，走勢為下降趨勢時，您先行作『低進或逢低買進』動作，那麼短期內 ，您可能沒有高價可賣出 。 <br>2.&nbsp;&nbsp; 應在多頭排列的上升趨勢途中『低進或逢低買進』，那麼短期內，您才可能有高價可賣出。 <br>3.&nbsp;&nbsp; 空頭排列的下降趨勢中的某日大漲，分析師們都會強烈建議次日『 逢低買進 』，但是，假如某日的大漲僅是【修正】，次日的逢低買進 ，則不就是再次套牢 ! 假如某日的大漲是【反彈 】或【回升】，次日的可能無法『逢低買進』的機會了 ! <br>「假如我剛才在某價位買進作多或賣出作空，我就賺了 X 錢」的說法或 <br>「幸好我剛才沒有在某價位買進作多或賣出作空，不然我就賠了 X 錢」的想法 ? <br>1.&nbsp;&nbsp; 通常投資人都會在前項狀況時，在『既成事實』的事後有這類【說法】，而在後項狀況時，會在『既成事實』的事後有這類【想法】，因為於『既成事實』時，走勢有如預期，則在事後有這類【說法】，若走勢沒有如預期，則在事後有這類【想法】，這是大部份人一種錯誤的心態，在剛才的某價位時，這些人是看不懂趨勢的方向的，因為他們在該位置有能力判斷趨勢的方向時，他一定實際執行買進或賣出的動作了。 <br>2.&nbsp;&nbsp; 短期趨勢的方向判斷錯誤是『常有的事』，既然自認行情走勢將如您預期的順向，就應當機立斷下單做買賣，或誠實記錄於紙上模擬操作中，無論事後的事實證明是對或錯 ，都要做《檢討》的動作，檢討【對】的判斷，做為日後重複【對】的判斷 ，檢討【錯】的判斷，做為日後不再犯錯，千萬不可有《假如》的說法或《幸好》的想法，否則您的操作技能將停留在【幻想】的階段。 <br>3.&nbsp;&nbsp; 多頭上升趨勢途中，若先《 高出 》之後，在當日內，可能有《 低進 》的機會 ，但機率不是很高。空頭下降趨勢途中，若先《 低進 》之後，在當日內 ，可能有《 高出 》的機會，但機率也不是很高。 <br>&nbsp; </p><p><br>長期、中期、短期 策略 <br>&nbsp;&nbsp;&nbsp; </p><p><br>長期、中期、短期 策略簡易說明： <br>1.&nbsp;&nbsp; ”長期”是指平均 2個半月以上左右進出一次。 <br>2.&nbsp;&nbsp; ”中期”是指平均 1個半月 ~ 2個月左右進出一次。 <br>3.&nbsp;&nbsp; ”短期”是指平均半個月 ~ 1個月左右進出一次。 <br>4.&nbsp;&nbsp; 通常依長期指標進出者不多，尤其股市，長期投資有時將持有越久虧越多。 <br>5.&nbsp;&nbsp; 一般而言，依短期指標配合中期指標進出即可。 <br>6.&nbsp;&nbsp; 短期指標對應於『日線』指標，中期指標對應於『週線』指標。 <br>&nbsp; </p><p><br>期貨與現貨之間，正價差與逆價差的觀察要領 <br>&nbsp; <br>期貨大於現貨 50左右 ( 正價差 ) <br>1、 例：期貨小漲 20 現貨中漲 60 <br>2、 例：期貨不漲 0 現貨小漲 40 </p><p>3、 例：期貨小跌 40 現貨不跌 0 <br>4、 例：期貨中跌 60 現貨小跌 20 </p><p><br>&nbsp; <br>期貨小於現貨 50左右 ( 逆價差 ) <br>1、 例：期貨中漲 60 現貨小漲 20 <br>2、 例：期貨小漲 40 現貨不漲 0 </p><p>3、 例：期貨不跌 0 現貨小跌 40 <br>4、 例：期貨小跌 20 現貨中跌 60 </p><p><br>&nbsp; <br>期貨大於現貨 100左右 ( 正價差 ) <br>1、 例：期貨小漲 40 現貨中漲 120 <br>2、 例：期貨不漲 0 現貨小漲 80 </p><p>3、 例：期貨小跌 80 現貨不跌 0 <br>4、 例：期貨中跌 120 現貨小跌 40 </p><p><br>&nbsp; <br>期貨小於現貨 100左右 ( 逆價差 ) <br>1、 例：期貨中漲 120 現貨小漲 40 <br>2、 例：期貨小漲 80 現貨不漲 0 </p><p>3、 例：期貨不跌 0 現貨小跌 80 <br>4、 例：期貨小跌 40 現貨中跌 120 </p><p><br>&nbsp; </p><p>循環操作策略： <br>位置順序： 峰之始 → KD形成黃金交叉 ( 在低檔向上交叉 ) → MA形成黃金交叉 ( 在低檔向上交叉 ) → 多頭上升趨勢 → 多頭上升趨勢 → 「峰循環」將至 → 峰之終 → 谷之始 → KD形成死亡交叉 ( 在高檔向下交叉 ) → MA形成死亡交叉 ( 在高檔向下交叉 ) → 空頭下降趨勢 → 空頭下降趨勢 → 「谷循環」將至 → 谷之終 <br>在倉部位順序： 多 3 空 3 → 多 3 空 2 → 多 3 空 0 → 低進高出 → 高出低進 → 多 3 空 1 → 多3空3 → 多 3 空 3 → 多 2 空 3 → 多 0 空 3 → 高出低進 → 低進高出 → 多 1 空 3 → 多 3 空 3 <br>策略順序： 扣鎖 → 多 1 → 多 3 → 觀望 → 觀望 → 多 2 → 扣鎖 → 扣鎖 → 空 1 → 空 3 → 觀望 → 觀望 → 空 2 → 扣鎖 <br>【日線】循環操作策略詳細說明： <br>&nbsp; </p><p>&nbsp; 3.&nbsp;&nbsp; 待 5MA與 10MA形成黃金交叉 ( 在低檔向上交叉 ) 之後，再次確認了多頭上升趨勢，在此位置再次〔融券買進 ( 回補 ) 2單位〕，即【作多 3單位】，作『多 3 空 0』的【多 3】策略；此位置的在倉部位為《 融資 50%、融券 0% 》。&nbsp;&nbsp; </p><p>&nbsp; 4.&nbsp;&nbsp; 完成了【作多 3單位】進場之後，只需留意波段「峰循環」的進度及完成與否，或某天的『跌』，是否會影響多頭上升趨勢，若否，則採取觀望策略，必要時可採取【 低進高出 】的短線策略，但《低進》之部位，務必於當日收盤前平倉，即留倉之《作多》部位 ，不可增加。 <br>註：此位置不可採取【高出低進】的短線策略，因多頭上升趨勢途中，若先《高出》之後，通常在短期內 ，可能沒有《低進》的機會。&nbsp;&nbsp; </p><p>&nbsp; 5.&nbsp;&nbsp; 待波段「峰循環」將至，在此位置﹝融券賣出 1單位﹞，即利用盤中高點初次建立『空 1』的部位，來扣鎖此波段《作多》之利潤，作『多 3 空 1』的【多 2】策略；此位置的在倉部位為《融資 50%、融券 15%》。&nbsp;&nbsp; </p><p>&nbsp; 6.&nbsp;&nbsp; 待波段「峰循環」完成之後，在此位置再次﹝融券賣出 2單位﹞，即利用盤中高點再次建立『 空 2 』的部位，來扣鎖此波段《作多》之利潤，作『多 3 空 3』的【扣鎖】策略；此位置的在倉部位為《融資 50%、融券 50%》。&nbsp;&nbsp; </p><p>&nbsp; 9.&nbsp;&nbsp; 待 20KD形成死亡交叉 ( 在高檔向下交叉 ) 之後，確認了空頭下降趨勢，在此位置〔融資賣出 1單位〕，即【作空 1單位】，作『多 2 空 3』的【空 1】策略；此位置的在倉部位為《融資 35% 、融券 50%》。&nbsp;&nbsp; </p><p>&nbsp; 10.&nbsp;&nbsp; 待 5MA與 10MA形成死亡交叉 ( 在高檔向下交叉 ) 之後，再次確認了空頭下降趨勢，在此位置再次〔融資賣出 2單位〕，即【作空 3單位】，作『多 0 空 3』的【空 3】策略；此位置的在倉部位為《融資 35%、融券 50%》。&nbsp;&nbsp; </p><p>&nbsp; 11.&nbsp;&nbsp; 完成了【作空 3單位】進場之後，只需留意波段「谷循環」的進度及完成與否，或某天的『漲』，是否會影響空頭下降趨勢，若否，則採取觀望策略，必要時可採取【高出低進 】的短線策略，但《高出》之部位，務必於當日收盤前平倉，即留倉之《作空》部位不可增加。 <br>註：此位置不可採取【低進高出】的短線策略，因空頭下降趨勢途中，若先《低進》之後，通常在短期內，可能沒有《高出》的機會。&nbsp;&nbsp; </p><p>&nbsp; 12.&nbsp;&nbsp; 待波段﹝谷循環﹞將至，在此位置〔融資買進 1單位〕，即利用盤中低點初次建立『多 1』的部位，來扣鎖此波段《作空》之利潤，作『多 1 空 3』的【空 2】 策略；此位置的在倉部位為《融資 15%、融券 50%》。&nbsp;&nbsp; </p><p>&nbsp; 13.&nbsp;&nbsp; 待波段﹝谷循環﹞完成之後，在此位置 ，再次〔融資買進 2單位〕，即利用盤中低點，再次建立『多 2』的部位，來扣鎖此波段《作空》之利潤，作『多 3 空 3』的【扣鎖】策略；此位置的在倉部位為《融資 50%、融券 50%》。&nbsp;&nbsp; </p><p>&nbsp; 14.&nbsp;&nbsp; 上述第 12 及 13 項所建立『多』的部位 ( 實際上 ，並未《作多》)，在此位置除了扣鎖此波段《作空》之利潤之外，並等待走勢確認進行下一波段的﹝峰循環﹞時，依確認訊號出現時，將《作空》的部位做『買進』，來賺取《作多》之利潤。&nbsp;&nbsp; </p><p>&nbsp; 15.&nbsp;&nbsp; 銜接至第 1項, 並依序循環操作。<br>
