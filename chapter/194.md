# 2013年公開持股報告書

```
這份報告主要的目的是想呈現五年期的投資紀錄，藉由公開透明化的
交易過程讓大家真實看到巴小智是如何看待股票投資與抱股票的耐心。

請各位不要去理會文章內容買了甚麼股票或買幾檔個股，這些都不是
本文強調的重點! 因為不是在跟各位報明牌，也不是在看短期績效，
而是想證明投資人抱股票的耐心是的確可以在長期 (五年~三十年期間)
獲得令人滿意的平均年化報酬率。
 面對未實現帳損相信是多數人在持有股票期間會經常遇到的情形，
但遇到了該如何自處呢?

我選擇相信當初對企業的認識與內在價值的判斷，不因為股票價格崩跌
而質疑它！因此在實際股票操作中，不會因股價下跌而隨意停損，
除非因為企業營運成績一再令人失望或企業經營階層做出不利於股東情事，
才會不計代價提前出場停損離開!

目前的投資心態已經將未實現帳損視為投資過程必經的歷程之一，
這是很正常的事情! 習慣它所帶來的考驗並無什麼好擔憂的。

自2012/09/03日開始第一次"試水溫"(少量買進股票)，但真正大量佈局
股票是在去年10月份，因此將9月底視為投資滿一年期的時間點，至於
五年後它可以得到多少累計報酬率？我們一起耐心觀察下去吧！

不管最後結果如何？本人都將忠實記錄這些投資過程，退休六年來
對股票資產最大的感觸是股票價格的波動性很大，如果時時刻刻都在
看盤，很在乎帳面損益，那投資股票肯定不會輕鬆的事。

彼得·林區說：『決定投資者投資成敗的關鍵在於投資者的耐心、
獨立精神、基本常識、對痛苦的忍耐力、坦率、超然、堅持不懈、
腦袋靈活、主動承認錯誤的精神，以及不受別人慌亂的影響，堅持到
投資獲得成功的能力。至關重要的是要能夠抵抗得了你人性的弱點。』

投資成果需要時間的累積,或許運氣好半年內，也許一年半也有可能
拉長到三年～五年不等，不管時間長或短？我們都要堅定“耐心、信心
與決心”因為股票的持有時間是投資人躲避股價短暫下跌最佳的武器。

過去六年來本人就是這樣在股市慢慢賺到錢的，也許有人急於想短期
獲利，投資步調相當積極屬於極為認真的投資風格，但本人愚昧沒有
這樣的超能力！若閣下是這類積極投資者請不要再看下去，因為我的
投資步伐相當無聊且無趣，且想保有優雅的投資風格！

為了讓網友們可以定期檢視帳面損益，這些個人投資紀錄表都會在
每個月底公布一次！而年度投資報酬率則在每年的9/30日公佈，
連續操作五年期。深感眾多散戶需要看到的是真實投資紀錄
而不僅僅是書上的那些深奧理論，理論若沒有實行也只是空談！
如果在這份報告內可以讓大家看到如何實現"用心於不交易"的執行力，
本人將會非常開心！這才是我想要呈現給大家明白的重點！
呵呵…


巴小智 2013/09/30

```
