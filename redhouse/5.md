# 用財務指標篩選、避開炒作公司。

我們關心的重點在於，能否用財務分析的角度避開這類公司？

```
1.     ROE近五年大於10%
2.     營業現金流量對稅後淨利大於50%
3.     自由現金流量近五年大於0
```
