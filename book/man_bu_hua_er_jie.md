# 漫步華爾街


<div class="article-content-inner" id="article-content-inner" itemprop="articleBody">
                <p>這本書號稱35年最經典的個人投資指南, 結合了投資理論及實務, 並佐以實證研究深入淺出地<br>
介紹投資理論且提供實用的投資理財法則, 是少有的理論與實務結合的理財書籍. &nbsp;作者Malkiel<br>
本身不但本身是經濟學者任教於美國普林斯頓大學, 並曾擔任數家大型投資公司的董事,&nbsp;也曾<br>
獲聘問美國總統經濟諮詢委員會委員, 更是一位終身投資且很成功的投資人. 兼具理論及實務<br>
的雄厚背景, 是本書能夠寫得如此精彩的主要理由.<br>
<br>
這本書自1973年出版以來, 歷經多次改版修訂, 不斷加入新的財務投資理論, 共分為四部:&nbsp;<br>

第三部: 學習新投資技術--介紹自50年代馬可維茲(Harry Markowitz)的投資組合理論到最近的<br>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 財務行為學<br>
第四部: 你也可以是理財高手--有作者建議的個人理財的十項演練, 美國股票及債券報酬歷史<br>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;及展望分析及做好人生不同階段投資規畫的建議<br>

由於書中關於泡沫歷史的幾章因之前有介紹貪婪時代一書已有著墨, 接下來幾篇介紹本書的<br>

<br>
<u><strong>隨機漫步</strong></u><br>
<br>
首先作者介紹何謂隨機漫步?&nbsp;<br>
<br>
<span style="font-family:新細明體,serif">"凡是未來發展方向不能依據過去行為加以推測者</span>, <span style="font-family:新細明體,serif">就是所謂隨機漫步</span>(Random Walk). <span style="font-family:新細明體,serif">用在<br>
股票市場上</span>, <span style="font-family:新細明體,serif">指的則是短期的股價變動無法預測</span>, <span style="font-family:新細明體,serif">理財顧問</span>, <span style="font-family:新細明體,serif">盈餘預測</span>, <span style="font-family:新細明體,serif">或複雜的圖表模型都幫<br>
不上忙</span>....<span style="font-family:新細明體,serif">說得極端一些</span>, <span style="font-family:新細明體,serif">隨機漫步意味著一隻蒙著眼睛的猴子</span>, <span style="font-family:新細明體,serif">對著報紙金融版擲鏢挑選出來<br>
的投資組合</span>, <span style="font-family:新細明體,serif">也會和專家挑選的組合一樣好</span>."<br>
<br>
學術界將隨機漫步理論分為三種型式: 弱式, 半強式, 強式<br>
<br>
<span style="font-family:新細明體,serif">1. 弱式市場</span>—<span style="font-family:新細明體,serif">我們不能由過去的股價來預測未來的股價</span><br>
<span style="font-family:新細明體,serif">2. 半強式市場</span>—<span style="font-family:新細明體,serif">已公開的資訊甚至不能用來推測未來價格</span><br>
<span style="font-family:新細明體,serif">3. 強式市場</span>—<span style="font-family:新細明體,serif">任何事</span>(<span style="font-family:新細明體,serif">甚至未公開的發展中事件</span>)<span style="font-family:新細明體,serif">都不能用以預測未來價格</span><br>
<br>
在<span style="font-family:新細明體,serif">弱式市場, 技術分析無效; 在半強式市場, 基本分析無效; 在強式市場內線交易無效</span><br>
<br>
學術界並依此創造出「新投資技術」跟華爾街專家抗衡. 但<span style="font-family:新細明體,serif">效率市場建立在幾項脆弱的假設上:</span><br>
<br>
<span style="font-family:新細明體,serif">1. 相信完美的價格存在</span><br>
<span style="font-family:新細明體,serif">&nbsp; &nbsp; --充份的例證顯示</span>, <span style="font-family:新細明體,serif">股價有時不依任何人的價值估計</span>, <span style="font-family:新細明體,serif">購買者也常有一窩蜂的狂熱行為</span><br>
<span style="font-family:新細明體,serif">2. 消息能在瞬間傳播</span><br>
<span style="font-family:新細明體,serif">&nbsp; &nbsp; --就算受機構投資人青睞的重要股票能迅速反映所有相關消息</span>, <span style="font-family:新細明體,serif">但是沒被專家緊密注意的數<br>
&nbsp; &nbsp; &nbsp; &nbsp;千家小公司卻很可能做不到</span>.<br>
<span style="font-family:新細明體,serif">3. 將已知的消息轉變為股票真實價值的估計不可靠也很難證實</span><br>
<br>
<span style="font-family:新細明體,serif">書中的一個比喻很傳神地比喻了效率市場的假說.</span><br>
<br>
<span style="font-family:新細明體,serif">有一位財務教授和他的學生一起散步</span>, <span style="font-family:新細明體,serif">他們看見地面上有一張</span>100<span style="font-family:新細明體,serif">美元鈔票</span>, <span style="font-family:新細明體,serif">學生停下來去撿</span>,<br>
<span style="font-family:新細明體,serif">但教授說</span>:<span style="font-family:新細明體,serif">「不用費事</span>, <span style="font-family:新細明體,serif">如果它真是一張</span>100<span style="font-family:新細明體,serif">美元鈔票</span>, <span style="font-family:新細明體,serif">早就不在這裡了</span>.<span style="font-family:新細明體,serif">」</span><br>
<br>
<span style="font-family:新細明體,serif">某些人同意地面上沒有</span>100<span style="font-family:新細明體,serif">美元美鈔可以撿</span>, <span style="font-family:新細明體,serif">但是更多人認為地面上不乏零錢</span>. <span style="font-family:新細明體,serif">作者認為</span><br>
<br>
<span style="font-family:新細明體,serif">"沒有人能持續成功的預測市場的走向</span>, <span style="font-family:新細明體,serif">或是個別股票的相對吸引力</span>, <span style="font-family:新細明體,serif">因此沒有人能持續獲得高<br>
於市場平均的報酬</span>. <span style="font-family:新細明體,serif">雖然市場上偶爾出現有利可圖的交易機會</span>, <span style="font-family:新細明體,serif">但當它們為人所知之後就失效</span>.<br>
<span style="font-family:新細明體,serif">無論個人或機構</span>, <span style="font-family:新細明體,serif">到現在為止</span>, <span style="font-family:新細明體,serif">都未能長期</span>, <span style="font-family:新細明體,serif">持續地找出經過風險調整</span>, <span style="font-family:新細明體,serif">有利可圖的股票交易機<br>
會</span>, <span style="font-family:新細明體,serif">特別是當它們需要付稅和交易手續費時</span>."<br>
<br>
換句話說, 作者認為市場偶爾會出現失靈, 因此你偶爾會看到地上有錢可撿, 但是這種機會不多.<br>
<br>
但是讓我們看看以下投資大師的報酬率紀錄</p>

<p>我相信確實是有人能持續獲得高於市場平均的報酬, 只不過跟幸福學一樣我們要研究的是這些<br>
高手而不是一般人的投資方法. 而財務行為學也打破了投資人是理性的假設, 因此柯斯托蘭尼說<br>
固執投資人因為有金錢,&nbsp;想法, 信念, 耐心, 運氣才能賺到盲從及無法戰勝情緒之猶豫投資人的錢.<br>
巴菲特也才能透過買進價值被低估的股票長期獲利. 可見投資人的不理性至少是效率市場的一<br>
大罩門, 但是隨著電子媒體愈來愈發達, 資訊及投資知識及方法的傳遞愈來愈快速, 市場還是會<br>
往效率的方向走, 但只要群眾還在, 人性不會改變, 就有獲利的機會.<br>
<br>
如果效率市場假說成立, 那一些投資專家都不用混了, 因此書中提到華爾街專家用基本分析或<br>
技術分析來武裝自己, 對抗學術界的炮火. 這本書主要便是介紹以下三種投資方法, 並以實證來<br>
說明其實用性, 再提出自己的投資建議及人生各階段的理財規劃.<br>
<br>
1. 基本分析--「磐石理論」(firm-foundation theory)<br>
2. 技術分析--「空中樓閣理論」(castle-in-the-air)<br>
3. 新投資技術--「資本資產訂價模式」(capital asset pricing model, CAPM) 及針對CAPM改<br>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; 善的「套利訂價理論」(artbitrage pricing theory, APT)<br>
<br>
下一篇我將介紹第一個投資方法--「磐石理論」</p>

<div class="article-content-inner" id="article-content-inner" itemprop="articleBody">
                <p><strong><u>「磐石理論」--代表人物巴菲特</u></strong><br>
<br>
<span style="font-family:新細明體,serif">"磐石理論認為</span>,&nbsp;<span style="font-family:新細明體,serif">不論是股票或房地產</span>,&nbsp;<span style="font-family:新細明體,serif">每一種投資工具都有確定的內在標準</span>--<span style="font-family:新細明體,serif">「真實價值」</span><br>
(intrinsic value),&nbsp;<span style="font-family:新細明體,serif">可以經由仔細分析目前和未來的展望而算出</span>.&nbsp;<span style="font-family:新細明體,serif">當市價跌破或漲過「真實價值」<br>
時</span>,&nbsp;<span style="font-family:新細明體,serif">買進或賣出的時機就到了</span>."<br>
<br>
所謂股票的真實價值就是這張股票能為我賺進的錢, 也就是股票每年配給我們的現金, 如果公<br>
司還一直沒倒, 那現金就叫做股利, 如果公司有一天倒了, 清算之後還有現金可拿, 那就是最後<br>
一筆現金了. &nbsp;另外, 因為現金有時間價值, 也就是今天存一塊錢, 明天有利息可拿. 因此每年的<br>
現金流量需要折現. 那用什麼利率折現? 一般是用預期報酬率或資金成本, 用預期報酬率或資<br>
金成本折現的意思就是如果未來每年的現金流量預估是如此, 而我的預期報酬率或我資金的<br>
成本如果要10%,&nbsp;那麼這張股票對我的價值或我願意出的價格就是股票未來每年幫我賺進的<br>
現金收入用10%折現的金額稱為現值.<br>
<br>
雖然作者為了讓本書寫得平易近人, 書中都不用財務管理及投資學那些嚇人的公式, 但為了方<br>
便說明我還是列一下公式.&nbsp;<br>
&nbsp;</p>

<div style="text-align: center;">&nbsp;</div>

<div style="text-align: center"><img loading="lazy" title="" style="border-width:0px; margin:0.7em 0px; float:left" alt="" src="http://pics18.yamedia.tw/41/userfile/w/wj2008/blog/1522de39c03e52.jpg"></div>

<p><br>
<br>
<br>
<br>
<br>
<br>
公式中的g是各期的成長率; D是目前的股利; k 是期望報酬率; P0就是現值也就是真實價值.<br>
<br>
<span style="font-family:新細明體,serif">為了估計股票的真實價值</span>,&nbsp;<span style="font-family:新細明體,serif">基本分析師最重要的工作是估計公司未來盈餘和股利的流量</span>.&nbsp;<span style="font-family:新細明體,serif">股票<br>
價值估算基礎為投資人可望從股票獲得的所有現金流量之現值或貼現值</span>,&nbsp;<span style="font-family:新細明體,serif">因此分析師必須估計<br>
公司的營業額</span>,&nbsp;<span style="font-family:新細明體,serif">營運成本</span>,&nbsp;<span style="font-family:新細明體,serif">公司稅稅率</span>,&nbsp;<span style="font-family:新細明體,serif">折舊政策</span>,&nbsp;<span style="font-family:新細明體,serif">以及資本額的來源和成本</span>.&nbsp;<span style="font-family:新細明體,serif">證券分析師必須<br>
研究公司過去記錄</span>,&nbsp;<span style="font-family:新細明體,serif">損益表</span>,&nbsp;<span style="font-family:新細明體,serif">資產負債表</span>,&nbsp;<span style="font-family:新細明體,serif">投資計畫</span>,&nbsp;<span style="font-family:新細明體,serif">加上親自拜訪</span>,&nbsp;<span style="font-family:新細明體,serif">評估公司管理陣容</span>.&nbsp;<span style="font-family:新細明體,serif">既然公<br>
司平日前景深受本身產業前景影響</span>,&nbsp;<span style="font-family:新細明體,serif">證券分析師最好能從產業前景下手分析</span>.&nbsp;<span style="font-family:新細明體,serif">每位基本分析師<br>
都希望經由透徹的產業研究</span>,&nbsp;<span style="font-family:新細明體,serif">掌握對未來影響</span>,&nbsp;<span style="font-family:新細明體,serif">但目前尚未反映到價格上的因素</span>.<br>
<br>
<span style="font-family:新細明體,serif">基本分析師利用四個基本決定因素來估計任何股票的適當價格</span><br>
&nbsp;<br>
<strong><span style="font-family:新細明體,serif">決定因素一</span>:&nbsp;<span style="font-family:新細明體,serif">預期</span>(<span style="font-family:新細明體,serif">股利</span>)<span style="font-family:新細明體,serif">成長率</span></strong><br>
<span style="font-family:新細明體,serif">股利成長率的複利效果驚人</span>, 15%<span style="font-family:新細明體,serif">的成長率代表股利每五年就增加一倍</span>.&nbsp;<span style="font-family:新細明體,serif">但陷</span><span style="font-family:新細明體,serif">阱在於股利成<br>
長不會永遠持續下去</span>,&nbsp;<span style="font-family:新細明體,serif">理由很簡單</span>:&nbsp;<span style="font-family:新細明體,serif">公司和產業也如同大多數</span><span style="font-family:新細明體,serif">生物一樣</span>,&nbsp;<span style="font-family:新細明體,serif">有生命的循環週期</span>.&nbsp;<span style="font-family:新細明體,serif">想<br>
想一百年前美國的頂尖公司現在都已經消</span><span style="font-family:新細明體,serif">失了</span>.&nbsp;<span style="font-family:新細明體,serif">就算能逃過自然的生命週期</span>,&nbsp;<span style="font-family:新細明體,serif">公司也會愈來愈<br>
難保持同樣的成長率</span>.<br>
&nbsp;<br>
<span style="font-family:新細明體,serif">磐石理論者評估股價的第一條規則</span><br>
<span style="font-family:新細明體,serif">規則一</span>:&nbsp;<span style="font-family:新細明體,serif">理性投資人願意為較高的股利和盈餘成長率付出較高的股價</span><br>
<span style="font-family:新細明體,serif">規則一的附加推論</span>:&nbsp;<span style="font-family:新細明體,serif">理性投資者願意為預期成長期較長的股票</span>,&nbsp;<span style="font-family:新細明體,serif">付出較高的</span><span style="font-family:新細明體,serif">股價</span><br>
&nbsp;<br>
<span style="font-family:新細明體,serif">我們把前項敘述的價格改為本益比</span>,&nbsp;<span style="font-family:新細明體,serif">問題會變成</span>:&nbsp;<span style="font-family:新細明體,serif">是否預期成長率較高</span>,&nbsp;<span style="font-family:新細明體,serif">實際的本益比也會比<br>
較高</span>?&nbsp;<span style="font-family:新細明體,serif">柯瑞格</span>(John Cragg)<span style="font-family:新細明體,serif">和作者的研究是肯定的</span>.<br>
<br>
<strong><span style="font-family:新細明體,serif">決定因素二</span>:&nbsp;<span style="font-family:新細明體,serif">預期股利的發放</span></strong><br>
<span style="font-family:新細明體,serif">在其他情況相同的條件下</span>,&nbsp;<span style="font-family:新細明體,serif">支付的股利愈高</span>,&nbsp;<span style="font-family:新細明體,serif">股票愈有價值</span>.<br>
<span style="font-family:新細明體,serif">規則二</span>:&nbsp;<span style="font-family:新細明體,serif">在其他情況相同的條件下</span>,&nbsp;<span style="font-family:新細明體,serif">理性的投資人願意付出較高的代價</span>,&nbsp;<span style="font-family:新細明體,serif">購買「支付現金股利<br>
占盈餘比例較高」的股票</span><br>
&nbsp;<br>
<strong><span style="font-family:新細明體,serif">決定因素三</span>:&nbsp;<span style="font-family:新細明體,serif">風險程度</span></strong><br>
<span style="font-family:新細明體,serif">依據一項知名的理論說法</span>:&nbsp;<span style="font-family:新細明體,serif">某家公司股價</span>(<span style="font-family:新細明體,serif">或包括股利在內的每年總收益</span>)<span style="font-family:新細明體,serif">相</span><span style="font-family:新細明體,serif">較於市場的擺盪<br>
幅度愈大</span>,&nbsp;<span style="font-family:新細明體,serif">風險(</span><span style="font-family:新細明體,serif">β)</span><span style="font-family:新細明體,serif">就愈大</span>.<br>
<span style="font-family:新細明體,serif">規則三</span>:&nbsp;<span style="font-family:新細明體,serif">在其他情況相同的條件下</span>,&nbsp;<span style="font-family:新細明體,serif">理性</span>(<span style="font-family:新細明體,serif">風險規避</span>)<span style="font-family:新細明體,serif">的投資人會願意為風險較低的股票</span>,&nbsp;<span style="font-family:新細明體,serif">付出<br>
較高的價格</span><br>
&nbsp;<br>
<strong><span style="font-family:新細明體,serif">決定因素四</span>:&nbsp;<span style="font-family:新細明體,serif">市場利率水準</span></strong><br>
<span style="font-family:新細明體,serif">如果利率夠高</span>,&nbsp;<span style="font-family:新細明體,serif">利息可以代替股票</span>,&nbsp;<span style="font-family:新細明體,serif">提供穩定收入</span>.<br>
<span style="font-family:新細明體,serif">規則四</span>:&nbsp;<span style="font-family:新細明體,serif">在其他情況相同的條件下</span>,&nbsp;<span style="font-family:新細明體,serif">理性投資人在利率愈低時</span>,&nbsp;<span style="font-family:新細明體,serif">願意付出的股價愈高</span><br>
&nbsp;<br>
<span style="font-family:新細明體,serif">但是, 雖然邏輯跟公式很簡單, 實際應用卻不簡單, 因此作者也提出三項警告:</span><br>
<br>
<span style="color:rgb(255, 0, 0)"><span style="font-family:新細明體,serif">警告一</span>:&nbsp;<span style="font-family:新細明體,serif">目前情況無法證實對未來的預期</span></span><br>
<span style="font-family:新細明體,serif">預測未來盈餘和股利是最困難的事</span>,&nbsp;<span style="font-family:新細明體,serif">因為這不僅需要經濟學家的知識跟技巧</span>,&nbsp;<span style="font-family:新細明體,serif">還要有心理學<br>
家的敏銳</span>.&nbsp;<span style="font-family:新細明體,serif">此外</span>,&nbsp;<span style="font-family:新細明體,serif">保持客觀也極其困難</span>,&nbsp;<span style="font-family:新細明體,serif">你不是過度樂觀就是過度悲觀</span>.<br>
&nbsp;<br>
<span style="color:rgb(255, 0, 0)"><span style="font-family:新細明體,serif">警告二</span>:&nbsp;<span style="font-family:新細明體,serif">無法由不確定的資料中算出精確的數字</span></span><br>
<span style="font-family:新細明體,serif">依磐石理論計算真實價值的精確數學公式</span>,&nbsp;<span style="font-family:新細明體,serif">倚賴的是靠不住的基礎</span>—<span style="font-family:新細明體,serif">對未來的預測</span>.&nbsp;<span style="font-family:新細明體,serif">原則上</span>,<br>
<span style="font-family:新細明體,serif">普通股的價值有其基本的不確定性</span>.&nbsp;<span style="font-family:新細明體,serif">萬能的上帝恐怕也不知道股票的本益比該是多少</span>.<br>
&nbsp;<br>
<span style="color:rgb(255, 0, 0)"><span style="font-family:新細明體,serif">警告三</span>:&nbsp;<span style="font-family:新細明體,serif">母鵝的成長不一定等於公鵝的成長</span></span><br>
<span style="font-family:新細明體,serif">市場是重視成長的</span>,&nbsp;<span style="font-family:新細明體,serif">而高成長率也總是和高本益比相伴相隨</span>,&nbsp;<span style="font-family:新細明體,serif">但問題是</span>:&nbsp;<span style="font-family:新細明體,serif">到底要為較高的成長<br>
多付多少</span>?&nbsp;<span style="font-family:新細明體,serif">不同的時間有不同的答案</span><br>
&nbsp;<br>
<span style="font-family:新細明體,serif">雖然基本分析看來合情合理而且合乎科學</span>,&nbsp;<span style="font-family:新細明體,serif">但是這種分析還是有三個瑕疵:</span></p>

<ol>
	<li><span style="font-family:新細明體,serif">資訊和分析可能不正確</span></li>
	<li><span style="font-family:新細明體,serif">證券分析師對「價值」的估計可能不當</span></li>
	<li><span style="font-family:新細明體,serif">市場可能不會回頭矯正錯誤</span>,&nbsp;<span style="font-family:新細明體,serif">而股價也可能始終未反映出估計的價值</span></li>
</ol>

<p><span style="font-family:新細明體,serif">「預測未來盈餘」是證券分析師存在的理由</span>, <span style="font-family:新細明體,serif">就像</span>&lt;&lt;<span style="font-family:新細明體,serif">機構投資人</span>&gt;&gt;<span style="font-family:新細明體,serif">所說</span>:<span style="font-family:新細明體,serif">「盈餘是關鍵</span>, <span style="font-family:新細明體,serif">永遠<br>
都是」</span><span style="font-family:新細明體,serif">分析師通常根據過去記錄以預測未來</span>. <span style="font-family:新細明體,serif">一位分析師告訴我</span>:<span style="font-family:新細明體,serif">「過去盈餘成長的記錄</span>, <span style="font-family:新細明體,serif">是<br>
預測未來盈餘成長最可靠的指標</span>.<span style="font-family:新細明體,serif">」</span><br>
&nbsp;<br>
那我們來看看這些分析師的預測表現如何?&nbsp;<br>
<br>
<span style="font-family:新細明體,serif">作者請華爾街從事基本分析的十九家最知名的公司</span>, <span style="font-family:新細明體,serif">請這些公司對一些大型的標準普爾五百<br>
大企業一年及五年期盈餘進行預測</span>, <span style="font-family:新細明體,serif">結果發現證券分析師的預測並不比直接用過去趨勢插補<br>
法得到的預測好到那裡</span>.<br>
&nbsp;<br>
<span style="font-family:新細明體,serif">由於五年成長率預估不良</span>, <span style="font-family:新細明體,serif">證券分析師往往坦承</span>, <span style="font-family:新細明體,serif">五年實在太遙遠了</span>, <span style="font-family:新細明體,serif">不容易準確預測</span>. <span style="font-family:新細明體,serif">他們<br>
覺得還是應該以未來一年盈餘預測來評斷才對</span>. <span style="font-family:新細明體,serif">結果他們一年期的預測比五年期還糟</span>.<br>
&nbsp;<br>
<span style="font-family:新細明體,serif">分析師又反擊說電子業和各種隨景氣循環起伏的公司盈餘</span>, <span style="font-family:新細明體,serif">素來難測</span>, <span style="font-family:新細明體,serif">但試了公用事業</span>, <span style="font-family:新細明體,serif">結果<br>
發現即使是穩定的公用事業</span>, <span style="font-family:新細明體,serif">他們的預測還很離譜</span>.<br>
&nbsp;<br>
<span style="font-family:新細明體,serif">哈佛大學的山瑞德</span>(Michael Sandretto)<span style="font-family:新細明體,serif">和麻省理工的米瑞默斯</span>(Sudhir Mikrishnamurthi), <span style="font-family:新細明體,serif">追<br>
蹤最常被研究的</span>1000<span style="font-family:新細明體,serif">家公司一年期預測</span>. <span style="font-family:新細明體,serif">每家公司的預測大致來自五或六位分析師</span>. <span style="font-family:新細明體,serif">這項結<br>
論也很駭人</span>—<span style="font-family:新細明體,serif">在這五年研究期間中</span>, <span style="font-family:新細明體,serif">分析師平均每年的錯誤率為</span>31.3%<br>
&nbsp;<br>
<span style="font-family:新細明體,serif">作者相信有五項因素可以說明何以分析師在預測未來時如此困難</span>:</p>

<div style="margin-top:0cm;margin-right:24.0pt;margin-bottom:0cm;margin-bottom:.0001pt;"><br>
<span style="font-family:新細明體,serif">&nbsp; &nbsp; &nbsp; &nbsp;1. 隨機事件的影響</span></div>

<div style="margin-top:0cm;margin-right:24.0pt;margin-bottom:0cm;margin-bottom:.0001pt;"><span style="font-family:新細明體,serif">&nbsp; &nbsp; &nbsp; &nbsp;2.「有創意」的會計程序</span>, <span style="font-family:新細明體,serif">創造出來令人存疑的盈餘</span><br>
<span style="font-family:新細明體,serif">&nbsp; &nbsp; &nbsp; &nbsp;3. 許多分析師本身能力不足</span></div>

<div style="margin-top:0cm;margin-right:24.0pt;margin-bottom:0cm;margin-bottom:.0001pt;"><span style="font-family:新細明體,serif">&nbsp; &nbsp; &nbsp; &nbsp;4. 最好的分析師往往轉任銷售部門或轉而擔任投資組合管理</span></div>

<div style="margin-top:0cm;margin-right:24.0pt;margin-bottom:0cm;margin-bottom:.0001pt;"><span style="font-family:新細明體,serif">&nbsp; &nbsp; &nbsp; &nbsp;5. 在擁有大量投資銀行業務的公司裡</span>, <span style="font-family:新細明體,serif">分析師面對利益衝突困境</span></div>

<p>&nbsp;<br>
<span style="font-family:新細明體,serif">加州大學的巴伯</span>(Brad Barber)<span style="font-family:新細明體,serif">調查發現受到分析師強烈買進建議的股票</span>, <span style="font-family:新細明體,serif">每月績效比市場平<br>
均值低了</span>3%, <span style="font-family:新細明體,serif">反而受到強烈賣出建議的股票</span>, <span style="font-family:新細明體,serif">表現比市場整體高出</span>3.8%.<br>
&nbsp;<br>
<span style="font-family:新細明體,serif">多項研究再次相當一致地顯示</span>: <span style="font-family:新細明體,serif">投資人購買共同基金的獲利</span>, <span style="font-family:新細明體,serif">並不比買進持有沒有管理的廣泛<br>
股票指數好</span>.<br>
&nbsp;<br>
<span style="font-family:新細明體,serif">那為什麼我們每年都可以看到基金績效排名總是顯示出許多基金的表現優於平均值?</span>&nbsp;<span style="font-family:新細明體,serif">問題是<br>
這些績效看不出持續力</span>.<br>
&nbsp;<br>
<span style="font-family:新細明體,serif">七</span>O<span style="font-family:新細明體,serif">年代前二十名基金</span>, <span style="font-family:新細明體,serif">績效比所有基金平均績效好將近一倍</span>(19%<span style="font-family:新細明體,serif">對</span>10.4%), <span style="font-family:新細明體,serif">在八</span>O<span style="font-family:新細明體,serif">年代期間</span><br>
, <span style="font-family:新細明體,serif">他們的表現卻比平均績效差</span>(11.1%<span style="font-family:新細明體,serif">對</span>11.7%). <span style="font-family:新細明體,serif">唯一例外是</span>, <span style="font-family:新細明體,serif">由傳奇人物彼得</span>.<span style="font-family:新細明體,serif">林區領軍的麥哲<br>
倫基金</span>, <span style="font-family:新細明體,serif">該基金在七</span>O<span style="font-family:新細明體,serif">和八</span>O<span style="font-family:新細明體,serif">年代皆表現優異</span>. <span style="font-family:新細明體,serif">彼得</span>.<span style="font-family:新細明體,serif">林區已在</span>1990<span style="font-family:新細明體,serif">年以四十六歲盛年退休</span>, <span style="font-family:新細明體,serif">我<br>
們因而永遠無法知道他是否還能繼續擊敗市場</span>.<br>
&nbsp;<br>
<span style="font-family:新細明體,serif">在</span>1994<span style="font-family:新細明體,serif">到</span>1999<span style="font-family:新細明體,serif">表現前二十名最熱門的基金平均報酬率超過</span>30%, <span style="font-family:新細明體,serif">遠高於整體市場的投資報酬<br>
率</span>. <span style="font-family:新細明體,serif">但在網路泡沫</span>2000<span style="font-family:新細明體,serif">年的前六年裡</span>, <span style="font-family:新細明體,serif">這些基金的表現比整體市場更糟</span>.<br>
&nbsp;<br>
<span style="font-family:新細明體,serif">加格里大學</span>(University of Calgary)<span style="font-family:新細明體,serif">的伍德渥</span>(Richard Woodward)<span style="font-family:新細明體,serif">及邱</span>(Jess Chua)<span style="font-family:新細明體,serif">教授研究<br>
指出</span>, <span style="font-family:新細明體,serif">持有股票做為長期投資標的比擇時操作更好</span>, <span style="font-family:新細明體,serif">因為在股票多頭的獲利大過空頭的損失</span>.<br>
<span style="font-family:新細明體,serif">他們的結論是</span>:<span style="font-family:新細明體,serif">擇時操作者判斷必須有</span>70%<span style="font-family:新細明體,serif">的正確率</span>, <span style="font-family:新細明體,serif">才可以勝過買進持有的投資人</span>. <span style="font-family:新細明體,serif">不過我可<br>
沒見過任何人對市場變動的預測有七成把握</span>.<br>
&nbsp;<br>
<span style="font-family:新細明體,serif">自麥哲倫基金退休的彼得</span>.<span style="font-family:新細明體,serif">林區</span>, <span style="font-family:新細明體,serif">以及著名的巴菲特都承認</span>, <span style="font-family:新細明體,serif">大多數投資人購買指數基金會比投<br>
資於積極管理的共同基金更為有利.</span><br>
<br>
<span style="font-family:新細明體,serif">其實, 也因為預測如此困難, 連大師巴菲特都利用只投資較易預測之穩健成長的公司和安全邊<br>
際二個做法來增加獲利的機率及預測錯誤的風險.&nbsp;<br>
<br>
既然預測如此困難, 投資人應該如何做? 作者建議投資人可以採用的穩健投資三原則:</span><br>
<br>
<strong><span style="font-family:新細明體,serif">規則一</span>: <span style="font-family:新細明體,serif">只買未來五年或五年以上</span>, </strong><span style="font-family:新細明體,serif"><strong>盈餘成長高於平均值的公司</strong></span><br>
<br>
<strong><span style="font-family:新細明體,serif">規則二</span>: </strong><span style="font-family:新細明體,serif"><strong>絕不付出高過真實價值的股票</strong></span><br>
<br>
<span style="font-family:新細明體,serif">雖然不可能判定確切的真實價值</span>, <span style="font-family:新細明體,serif">但大略判斷股價是否合理還是做得到的</span>. <span style="font-family:新細明體,serif">一般說來</span>, <span style="font-family:新細明體,serif">市場平<br>
均本益比大致是個有用的基準</span>. <span style="font-family:新細明體,serif">作者建議的策略是</span>, <span style="font-family:新細明體,serif">購買尚未被發現</span>, <span style="font-family:新細明體,serif">本益比還沒有高出市場<br>
水準的成長股</span>. <span style="font-family:新細明體,serif">當然</span>, <span style="font-family:新細明體,serif">預測成長相當困難</span>. <span style="font-family:新細明體,serif">但只要買進時本益比低</span>, <span style="font-family:新細明體,serif">就算成長未實現且盈餘下跌</span><br>
, <span style="font-family:新細明體,serif">損失也有限</span>; <span style="font-family:新細明體,serif">但如果預測成真</span>, <span style="font-family:新細明體,serif">就有雙重好處了</span>.<br>
<br>
<span style="font-family:新細明體,serif">彼得</span>.<span style="font-family:新細明體,serif">林區比較股票的成長率和本益比</span>, <span style="font-family:新細明體,serif">只購買成長率比本益比大的股票</span> (PE/G ratio <span style="color:#ff0000">&lt;</span> 1)*<br>
<br>
<strong><span style="font-family:新細明體,serif">規則三</span>: <span style="font-family:新細明體,serif">尋找有題材的成長股</span>,</strong><span style="font-family:新細明體,serif"><strong>讓投資人建造空中樓閣</strong></span><br>
<br>
<span style="font-family:新細明體,serif">就是自問</span>, <span style="font-family:新細明體,serif">你的股票是否有故事題材可以滿足大眾的想像</span>. <span style="font-family:新細明體,serif">它是否能激發有感染力的夢想</span>?<br>
<span style="font-family:新細明體,serif">是否能讓投資人建立空中樓閣</span>? <span style="font-family:新細明體,serif">空中樓閣是否能建立在磐石上</span>?&nbsp;<span style="font-family:新細明體,serif">規則三不需要技術分析專家<br>
的技巧</span>, <span style="font-family:新細明體,serif">只需憑直覺或投機意識</span>, <span style="font-family:新細明體,serif">判斷股票的故事是否能吸引群眾</span>—<span style="font-family:新細明體,serif">特別是機構投資人</span><br>
<br>
<br>


<div class="article-content-inner" id="article-content-inner" itemprop="articleBody">
                <p><strong><u>「空中樓閣理論」--代表人物凱因斯</u></strong><br>
<br>
<span style="font-family:新細明體,serif">空中樓閣的投資理論強調心理因素</span>. <span style="font-family:新細明體,serif">凱因斯</span>(John Maynard Keynes)&nbsp;<span style="font-family:新細明體,serif">認為專業投資人不該<br>
把精力用於估算股票的真實價值</span>, <span style="font-family:新細明體,serif">而該用於分析大眾未來的動向</span>,&nbsp; <span style="font-family:新細明體,serif">以及在樂觀時期他們會<br>
如何把希望建築為空中樓閣</span>. <span style="font-family:新細明體,serif">成功的投資人最喜歡搶先機</span>, <span style="font-family:新細明體,serif">推測哪種情況最容易讓大家建<br>
築空中樓閣</span>, <span style="font-family:新細明體,serif">然後比眾人早一步下手</span>.&nbsp;<br>
<br>
<span style="font-family:新細明體,serif">凱因斯用英國同胞容易了解的名詞來說明</span>: <span style="font-family:新細明體,serif">股票遊戲就好比當報紙選美的評審一般</span>, <span style="font-family:新細明體,serif">你必<br>
須由上百張照片中挑出六張最漂亮的臉孔</span>, <span style="font-family:新細明體,serif">選出的結果最接近全體評審的</span>, <span style="font-family:新細明體,serif">就是贏家</span>.&nbsp;<span style="font-family:新細明體,serif">聰<br>
明的玩家都知道</span>, <span style="font-family:新細明體,serif">個人的審美偏好在競賽中無關緊要</span>, <span style="font-family:新細明體,serif">較聰明的做法是挑選其他人會選<br>
的</span>. <span style="font-family:新細明體,serif">因此最佳策略不是挑選你認為最美的</span>, <span style="font-family:新細明體,serif">而是推測一般人會推測的一般意見</span>.<br>
&nbsp;<br>
<span style="font-family:新細明體,serif">選美的推理代表了空中樓閣理論中價格評定的基本形式</span>. <span style="font-family:新細明體,serif">對買主而言</span>, <span style="font-family:新細明體,serif">一項投資值得該售<br>
價</span>, <span style="font-family:新細明體,serif">是因為他預期能夠以更高的價格售出</span>. <span style="font-family:新細明體,serif">新買主當然也預期未來會有人以更高的價格接手</span>.<br>
&nbsp;<br>
<span style="font-family:新細明體,serif">在空中樓閣的世界裡</span>, <span style="font-family:新細明體,serif">每分鐘都有乳臭未乾的傻瓜誕生</span>, <span style="font-family:新細明體,serif">願付更高的價格購買你的投資</span>.<br>
<span style="font-family:新細明體,serif">因此</span>, <span style="font-family:新細明體,serif">只要別人願意出高價</span>, <span style="font-family:新細明體,serif">價格根本不是問題</span>. <span style="font-family:新細明體,serif">這沒什麼道理可言</span>, <span style="font-family:新細明體,serif">純粹是大眾心理</span>. <span style="font-family:新細明體,serif">聰<br>
明的投資人要做的</span>, <span style="font-family:新細明體,serif">就是比別人搶先一步</span>. <span style="font-family:新細明體,serif">這個理論說難聽一點</span>, <span style="font-family:新細明體,serif">可以稱為「比傻」理論</span>.<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;<br>
<span style="font-family:新細明體,serif">摩根斯坦</span>(Oskar Morgenstern)<span style="font-family:新細明體,serif">是頭號鼓吹者</span>, <span style="font-family:新細明體,serif">他主張</span>, <span style="font-family:新細明體,serif">尋找股票的真實價值好比大海撈<br>
針</span>. <span style="font-family:新細明體,serif">他認為每位投資人的桌前都應該張貼下面這句拉丁格言</span>:<br>
&nbsp;<br>
“<span style="font-family:新細明體,serif">物品的價值所在</span>, <span style="font-family:新細明體,serif">就是他人願意支付的價格</span>”<br>
<br>
<span style="font-family:新細明體,serif"><strong><u>技術分析</u></strong><br>
<br>
技術分析是信仰中樓閣者用來預測適當買賣時機的方法.&nbsp;</span><span style="font-family:新細明體,serif">技術分析的首要原則是</span>: <span style="font-family:新細明體,serif">一家<br>
公司所有關於盈餘</span>, <span style="font-family:新細明體,serif">股利</span>, <span style="font-family:新細明體,serif">未來表現的資訊</span>, <span style="font-family:新細明體,serif">都已反映在過去市場股價上</span>. <span style="font-family:新細明體,serif">第二個原則是</span>,<br>
<span style="font-family:新細明體,serif">價格循「趨勢」變動</span>: <span style="font-family:新細明體,serif">上漲的股票繼續上漲</span>, <span style="font-family:新細明體,serif">休息的繼續休息</span>.<br>
&nbsp;<br>
<span style="font-family:新細明體,serif">股市狀況最難的問題也許是</span>: <span style="font-family:新細明體,serif">為什麼畫圖形會有用</span>? <span style="font-family:新細明體,serif">作者認為對技術分析較合理的解釋</span>.<br>
<span style="font-family:新細明體,serif">趨勢持續的可能原因有二</span>, <span style="font-family:新細明體,serif">第一是群眾心理的直覺天性</span>. <span style="font-family:新細明體,serif">當投資人看到熱門股節節上漲</span>,<br>
<span style="font-family:新細明體,serif">就急著想加入抬轎</span>, <span style="font-family:新細明體,serif">所以漲價本身在未來漲價的預言背後推一把</span>. <span style="font-family:新細明體,serif">每一次上漲都再次刺<br>
激投資人的胃口</span>, <span style="font-family:新細明體,serif">期待更上層樓</span>.<span style="font-family:新細明體,serif">第二是因為公司基本資訊取得不公平</span>, <span style="font-family:新細明體,serif">某些有利消息發<br>
生時</span>, <span style="font-family:新細明體,serif">傳說總是內部人士最早知道並買進</span>, <span style="font-family:新細明體,serif">造成價格上漲</span>. <span style="font-family:新細明體,serif">然後是內部人士的朋友</span>, <span style="font-family:新細明體,serif">專業<br>
人士</span>, <span style="font-family:新細明體,serif">機構法人</span>, <span style="font-family:新細明體,serif">最後才是一般投資人</span>. <span style="font-family:新細明體,serif">圖形分析師相信</span>, <span style="font-family:新細明體,serif">即使他們沒有內線消息</span>, <span style="font-family:新細明體,serif">但光靠<br>
觀察價格的變動</span>, <span style="font-family:新細明體,serif">他們也能嗅到「聰明錢」的氣味</span>, <span style="font-family:新細明體,serif">比大眾早好幾步進場</span>.<br>
&nbsp;<br>
<font face="新細明體, serif">在說明完技術分析的主要論點後, 作者開始引用許多的實證說明技術分析的作用甚微.</font><br>
<span style="font-family:新細明體,serif">作者曾要求學生實驗以</span>50<span style="font-family:新細明體,serif">美元為起點</span>, <span style="font-family:新細明體,serif">每個交易日的收盤價都以擲銅板來決定</span>, <span style="font-family:新細明體,serif">如果擲<br>
出正面</span>, <span style="font-family:新細明體,serif">收盤價便較前一交易日高半點</span>, <span style="font-family:新細明體,serif">擲出背面則較前一日跌半點</span>. <span style="font-family:新細明體,serif">結果這個隨意擲<br>
銅板而來的圖形和正常股價圖非常相似</span>, <span style="font-family:新細明體,serif">甚至也有漲跌循環</span>. <span style="font-family:新細明體,serif">在另一個擲銅板的實驗中<br>
還找出頭肩圖</span>, <span style="font-family:新細明體,serif">三重頂</span>, <span style="font-family:新細明體,serif">三重底</span>, <span style="font-family:新細明體,serif">以及其他更難懂的股票線形</span>. 作者用此實驗說明股價走<br>
勢是跟擲銅板一樣是隨機的沒有所謂的循環週期這回事.<br>
<br>
我認為這個實驗還是不能完全證明只能說是可能, 因為1+4=5, 不代表5只能是1+4, 也<br>
可以是2+3, 擲銅板跟股價走勢相似不代表股價走勢就一定向擲銅板一樣是隨機漫步<br>
沒有循環或趨勢. &nbsp;而且作者曾提到<br>
<br>
"<span style="font-family:新細明體,serif">對數學家而言</span>, <span style="font-family:新細明體,serif">股票圖形記下一連串數字和模擬股價實驗沒有多大差別</span>. <strong><span style="color:red"><span style="font-family:新細明體,serif">唯一的例外<br>
是</span></span><span style="color:red">, </span></strong><strong><span style="color:red"><span style="font-family:新細明體,serif">大多數股價平均指數的長期上漲趨勢都伴隨著盈餘和股利的長期成長</span></span><span style="color:red">.</span></strong> <span style="font-family:新細明體,serif">調整<br>
這個因素後</span>, <span style="font-family:新細明體,serif">就沒有實質差異了</span>."<br>
<br>
代表股價還是受盈餘和股利的影響, 就像柯斯托蘭尼舉的人遛狗的例子, 狗(股價)再怎<br>
麼跑, 還是不能脫離人(經濟)的繩子. <span style="color:#ff0000"><strong>股價受盈餘和股利影響, 公司盈餘受景氣影響,<br>
經濟景氣有循環, 股價走勢當然也會跟景氣循環有關. </strong></span>因此我覺得作者說股價沒有<br>
循環週期有點武斷, 我個人認為整體的股價指數有循環, 但循環期間不那麼固定.<br>
<br>
接下來作者又舉了以下幾個較普遍的技術分析手法及其有用性的實證:<br>
&nbsp;<br>
<span style="font-family:新細明體,serif">1. 過濾系統</span>(Filter System)<span style="font-family:新細明體,serif">假設</span>, <span style="font-family:新細明體,serif">當股價已跌至低點並開始攀升達到某個百分比</span>(<span style="font-family:新細明體,serif">例如</span><br>
5%), <span style="font-family:新細明體,serif">就可算是處於上升趨勢中</span>. <span style="font-family:新細明體,serif">股價若自高點反轉下跌</span>5%, <span style="font-family:新細明體,serif">就是在下</span><span style="font-family:新細明體,serif">降趨勢中</span>. <span style="font-family:新細明體,serif">這時<br>
候投資人應該在股票自低點上升</span>5%<span style="font-family:新細明體,serif">時買進並持有</span>, <span style="font-family:新細明體,serif">等股票抵達下一個高點</span>, <span style="font-family:新細明體,serif">並反轉下<br>
跌</span>5%<span style="font-family:新細明體,serif">時賣出</span>.<span style="font-family:新細明體,serif">各種過濾規則曾經一再被測試</span>, <span style="font-family:新細明體,serif">設定的漲跌幅自</span>1%<span style="font-family:新細明體,serif">至</span>50%<span style="font-family:新細明體,serif">都有</span>, <span style="font-family:新細明體,serif">時間涵<br>
蓋不同時期</span>, <span style="font-family:新細明體,serif">對象則從個股到各種股價平均指數</span>. <span style="font-family:新細明體,serif">結果再次顯示</span>, <span style="font-family:新細明體,serif">考慮了高額佣金後</span>,<br>
<span style="font-family:新細明體,serif">使用這套辦法買個股或股票指數</span>, <span style="font-family:新細明體,serif">並不一定能勝過簡單的買進持有策略</span>.<br>
&nbsp;<br>
<span style="font-family:新細明體,serif">2. 道式理論的基本假定是</span>, <span style="font-family:新細明體,serif">股市走高</span>, <span style="font-family:新細明體,serif">漲破前次高點買進</span>, <span style="font-family:新細明體,serif">走低跌破前次谷底時就要賣<br>
出,&nbsp;</span>也就是壓力和支撐的買賣策略.&nbsp;<span style="font-family:新細明體,serif">和買進持有具有市場性股票的簡單策略相比</span>, <span style="font-family:新細明體,serif">採用<br>
道式理論的表現反而較差</span>, <span style="font-family:新細明體,serif">因為它必須多付買賣佣金及手續費</span>.<br>
&nbsp;<br>
<span style="font-family:新細明體,serif">3. 相對強勢策略</span>(Relative-Strength System)<span style="font-family:新細明體,serif">是投資人買進並持有</span>”<span style="font-family:新細明體,serif">表現良好</span>”, <span style="font-family:新細明體,serif">優於市<br>
場整體表現</span>”<span style="font-family:新細明體,serif">的強勢股</span>, <span style="font-family:新細明體,serif">避免持有甚至放空</span>”<span style="font-family:新細明體,serif">表現較市場整體差</span>”<span style="font-family:新細明體,serif">的股票</span>. <span style="font-family:新細明體,serif">雖然有些時段<br>
相對強勢策略的績效的確優於買進持有策略</span>, <span style="font-family:新細明體,serif">但是沒有證據顯示優勢能持續保持</span>. <span style="font-family:新細明體,serif">再<br>
我們以電腦測試二十五年的資料後顯示</span>, <span style="font-family:新細明體,serif">扣除佣金後</span>, <span style="font-family:新細明體,serif">這類規則對投資人並無大助益</span>.<br>
&nbsp;<br>
<span style="font-family:新細明體,serif">4. 閱讀圖形</span>, <span style="font-family:新細明體,serif">有一項研究</span>, <span style="font-family:新細明體,serif">電腦匯出紐約證券交易所得</span>548<span style="font-family:新細明體,serif">種股票五年內的表現圖形</span>.<br>
<span style="font-family:新細明體,serif">電腦掃瞄所有圖形</span>, <span style="font-family:新細明體,serif">辨識出</span>32<span style="font-family:新細明體,serif">種最受歡迎的線形</span>; <span style="font-family:新細明體,serif">例如</span>, <span style="font-family:新細明體,serif">頭肩形</span>, <span style="font-family:新細明體,serif">三重頂或三重底</span>, <span style="font-family:新細明體,serif">槽形</span><br>
, <span style="font-family:新細明體,serif">楔形</span>, <span style="font-family:新細明體,serif">菱形等</span>. <span style="font-family:新細明體,serif">只要出現多頭線形就記錄買進</span>, <span style="font-family:新細明體,serif">反之就記錄賣出</span>. <span style="font-family:新細明體,serif">結果再度顯示</span>, <span style="font-family:新細明體,serif">技術<br>
分析的信號和接下來的股價表現沒以關係</span>. <span style="font-family:新細明體,serif">如果你跟著信號買進賣出</span>, <span style="font-family:新細明體,serif">扣掉手續費</span>,<br>
<span style="font-family:新細明體,serif">報酬仍不比買進持有好</span>.<br>
&nbsp;<br>
<span style="font-family:新細明體,serif">最後作者做了一個我認為較客觀的結論: <span style="color:#ff0000"><strong>"我沒有說技術分析的策略從來不賺錢</strong></span></span><span style="color:#ff0000"><strong>,<br>
<span style="font-family:新細明體,serif">相反地</span>, <span style="font-family:新細明體,serif">它們經常賺錢</span>. <span style="font-family:新細明體,serif">重點是</span>, <span style="font-family:新細明體,serif">簡單的「買進並持有」一樣賺錢</span>, <span style="font-family:新細明體,serif">甚至賺更多</span>. "</strong></span><br>
<br>
我自己在學校唸過財務管理及投資學, 本來也只相信基本分析, 但在我閱讀了一些投<br>
資書籍之後, 對技術分析的看法是: 技術分析的工具很多, 有些有用, 有些沒有, 有些<br>
在高點或低點特別有用, 有些要搭配著用, 有些要看情況用. 但不需要用太複雜的技<br>
術分析工具, 因為技術分析是歸納過去的歷史經驗, 歸納法無法普遍化, 太複雜花時<br>
間所增加的邊際效用不見得大, 不見得更準. 另外就是技術分析長線比短線, 大盤比<br>
個股更有參考價值. 但不管怎樣, 要使用一種投資方法之前, 至少要經過自己用歷史<br>
資料做回溯驗證, 還有就是該做停損要做停損, 如果你不像邵康節一般可用梅花易數<br>
預知未來的話.&nbsp;<br>
<br>
但我相信傑西李佛摩下面這句話<br>
<br>
" 華爾街和股票投機從來沒有發生過任何新鮮事。過去已經發生過的事情將來還會<br>
再次發生，一次又一次。因為人性沒有改變，人類的情緒深植在人類的本性之中，<br>
它們總是礙著人類的智慧。關於這一點，我十分確信"<br>
<br>
既然人性不變的貪婪跟恐懼會在投資歷史中一次又一次重演, 我相信股市是有可能<br>
出現重複的型態(pattern), 關鍵在於用什麼方法, 如何去辨認及如何應用來為自己賺<br>
進財富.<br>
